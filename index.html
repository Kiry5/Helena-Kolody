HTML

A Linguagem de Marcação de Hipertexto (HTML) é a linguagem principal da WEB. Conhecida em inglês como Hypertext Markup Language, ela permite que os desenvolvedores desenhem como serão exibidos pelo navegador os elementos da página WEB, como texto, hiperlinks e arquivos multimídia.

CSS

Ao falarmos sobre o que é CSS (Cascading Style Sheet, ou Folha de Estilo em Cascatas), é necessário dizer que sua aplicação é utilizada na estilização de componentes escritos em linguagens de marcação. Imagine que o HTML é a parede e o CSS fosse a tinta da parede.
